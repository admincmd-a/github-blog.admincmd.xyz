使用虚拟机对实体机硬盘安装操作系统

发表于2024-06-10|更新于2024-10-10|教程Windows 安装
|字数总计:1.7k|阅读时长:5分钟|阅读量:

本文需使用以下材料

  • Windows 安装镜像

  • VMware 虚拟机软件

  • 目标计算机（可以无显卡，必须有网卡 无线网卡 ）

  • 安装计算机（正常，可以跑虚拟机 4逻辑处理器 8GB RAM）

  • 一张可以装系统盘的硬盘

  • 一定动手能力

如果您使用此方法，请务必备份好数据，操作后无法恢复，

未备份数据，造成硬盘内文件丢失，作者不承担任何责任。

按照下列方法，如您继续，则默认您同意我们的声明。

建议你先将文章过一遍，这样可以更清楚的知道自己下一步要干什么。

注意：本文配图与代码框内有出入，请以围栏代码框内为准

前絮

这台电脑是我家里的一台老电脑，已经很久没有使用了，但是我想在上面安个 Windows Server，整个服务器使

这台电脑没有任何东东，除了一张PCI网卡，没有任何其他的东西（对，TM 连显卡都没有），所以装系统肯定会有很多麻烦根本就没法装系统。

准备工作

首先,我们要找台可以跑虚拟机的电脑
然后找张固态盘（机械盘也行，不过——太——慢——了）,拉来装系统
然后在准备一个系统镜像,可以从 这个页面 下载。
其他东西应该不用说吧

安装系统

如果想直接在磁盘上进行分区并应用镜像,其参考本文章

首先我们需要准备一个虚拟机，这里使用 VMware Workstation

创建虚拟机

装 VMware
打开 VMware Workstation，选择新建虚拟机
上面这几步应该不用说吧？

  1. 到安装操作系统这一步,请在安装程序光盘映像文件这一栏选择你选下载的系统镜像

  2. 客户端系统自己选你的版本

  3. 名称和位置随意

  4. 固件类型,一定要注意,以你要安装的电脑相符

    一定要注意,以你要安装的电脑相符,否则操作系统 99% 无法启动

    一般来讲,uefi 的电脑,其安全启动默认开启,所以是 uefi 的电脑最好把安全启动也给勾上,免得出一些奇怪的问题

    如果你的电脑比较老,比如是 11 年的,那基本确定是 BIOS ,如果电脑很新,比如是 21 年的,那基本确定是 UEFI

    不过如果你不知道,自己去问度娘,或者去bing找一下

  5. 处理器,以你跑虚拟机的电脑为准
    内存,以你跑虚拟机的电脑为准
    网络适配器,默认就行
    I/O 控制器,默认就行
    虚拟磁盘类型,按你准备的盘插口决定其实无所谓,选个 SATA 就行

  6. 磁盘请选使用物理磁盘啊这一步挺重要的

  7. 设备
    这一步有点复杂

很简单,一个一个猜去

......

  1. 按下 Win + R 打开运行框

  2. 输入diskmgmt.msc回车打开磁盘管理

  3. 然后你会看到你的磁盘,左边有一栏,写这磁盘 0等等等等,找到你要安装的盘,记下它的编号,比如磁盘 2

  4. 然后回到 VMware Workstation,依照上面的例子,你的磁盘就是PhysicalDrive 2

磁盘 3 就是 PhysicalDrive 3 等等

没有就退出重进

  1. 使用情况就选整个磁盘即可

其他的不用管，下一步完成即可

安装系统

  1. 启动虚拟机,进入安装程序

一般来说,vmware 都会自动识别系统镜像并加载无人值守程序,所以这一步应该不用你操心

MBR、GPT 分区表转换

安装过程出现以下情况,请按照提示操作:

说什么 GPT 分区表，什么 mbr 分区表不适用什么 EFI 啥的

以下操作位于虚拟机。

进行该操作之前,请先备份好数据,因为操作后无法恢复

若要将磁盘转换为 GPT/MBR 磁盘,请在转换磁盘前备份数据，因为会删除所有分区或卷。

  1. 按下 Shift + F10 打开命令提示符
  2. 输入DISKPART回车进入磁盘管理命令行
  3. 输入list disk查看磁盘列表
  4. 输入select disk 0选择第一个磁盘
  5. 输入clean清除所有分区

若要删除受保护的磁盘（如类型为“系统”与“保留”），则需要在 Delete Partition 命令后添加 Override 参数
如果需要一次性将磁盘上所有分区删除，请使用 Clean 命令

例如直接使用Delete Partition命令删除则会返回
虚拟磁盘服务错误:
如果不设置强制保护参数设置，则无法删除受保护的分区

而使用Delete Partition Override命令删除则会返回
DISKPART 成功删除了所选分区

如果是 mbr 分区表,不适用 efi
请参考

  1. 输入convert gpt转换为 gpt 分区表

如果是 GPT 分区表,说什么不适用 legacy,请参考

  1. 输入convert mbr转换为 mbr 分区表

然后
7. 输入exit退出命令行

下一步即可

然后等就可以了！

系统装完了？ou，那就下一步吧

配置系统

Windows Server 需要你在初次启动时设置内置管理员密码，对密码安全性要求很高，

本次装系统是在一台没有任何外围设备的电脑上，所以需要在装系统后在虚拟机里配置好系统。

此处省略该段，以你使用它的方式决定，列如允许远程连接、自定启动项……

物理机调配

把跑虚拟机的电脑关机

将你插上去的那张盘拔下来，咋装的咋拔
然后将盘插回之前的那台电脑

开机
要检测你的电脑是否已进入系统，有一个很简单的方法

把网线插入到电脑，看看指示灯是否亮起

请确保网卡驱动已正确安装！网线已正确插入
虚拟机内好了不代表现在就好

如果没好……

将电脑关机

把除系统盘以外的盘全部断开数据线连接
好，再开机

把网线插入到电脑，看看指示灯是否亮起

请确保网卡驱动已正确安装！网线已正确插入

把电脑砸了！

找张显卡吧……

网卡指示灯亮起就OK了
你现在可以用 IP 地址或者计算机名（可以提前将计算机改成SERVER之类的）进行访问了。

好了，就这样了。
